# 新規導入技術
- setInterval(function,msec)<br>第一引数に関数名、第2引数にミリ秒を指定する<br>第一引数で指定された関数を第２引数で指定された周期で実行する。例えば第二引数で1000を指定すれば1秒ごとに実行される
# 理解不足だった箇所
- 変数のスコープに対する理解
- setIntervalの第一引数
# 学び
- 関数の外の地の領域で記述された変数はグローバル変数となり、メモリ上に保存される
- 対して、関数内で定義された変数はローカル変数となり、処理が関数の外に出たときにメモリ上に保存されない
- setIntervalの第一引数には関数名のみを記述する
- 関数名()まで書いてしまうと、その場で実行した結果を渡すことになってしまう。
